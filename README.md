# LynaBot
Lyna Bot for Discord

Functions:
- entertainment(roll dice and pick random number)
- send random images
- manage channel
- play songs(experimental, in development)

# Installation

- place ffmpeg into tools folder
```
pip install -r (path to requirements.txt)
```

# Starting bot

Run bot:
```
python run_bot.py
```

Run server(Flask) with bot:
```
python start.py
```
